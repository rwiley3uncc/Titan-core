# Titan Build Plan

## Mission
Titan is Ron's personal assistant for school and life. It should reduce overload, keep work visible, and help Ron stay ahead instead of scrambling near deadlines.

## Product Principles
- One combined view for school and life
- Keep the sitrep simple, repeatable, and visible
- Suggest study blocks before auto-placing them
- Start big work early and create buffer before deadlines
- Prefer short, actionable output over long explanations
- Support TBI needs by reducing clutter and surfacing only what matters now

## Current Defaults
- Morning sitrep time: 08:00
- Study block length: 30 to 45 minutes
- Delay after getting home/eating: none
- Scheduling mode: suggest first
- Calendar source: Outlook (Mousenutts@live.com)
- School source: UNC Charlotte Canvas ICS feed
- Sitrep scope: school + life together

## MVP Goals
1. Import school dates/events from Canvas ICS
2. Read life calendar availability from Outlook
3. Generate a single morning sitrep
4. Suggest work blocks around real schedule
5. Start large assignments about a week early when possible
6. Create review/practice suggestions before tests
